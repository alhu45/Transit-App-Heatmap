# TTC Ridership Heatmap 
Predictive crowding visualization for Toronto subway stations

This project predicts TTC subway station ridership and visualizes crowding as a heatmap UI. The system automates data ingestion, ML-based prediction, and deployment using modern DevOps tooling.

## Features

- Interactive heatmap UI (React + Leaflet)
- Feature Engineering to prepare ML Training
- Machine learning ridership prediction (FastAPI + Python)
- Containerized frontend + backend (Docker)
- Fully deployed on AWS using Terraform (EKS + ECR)
- Automated CI/CD with GitHub Actions
- Scalable backend served with Kubernetes
